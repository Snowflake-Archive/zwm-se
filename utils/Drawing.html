Drawing

Utilities for drawing things (borders and sub-pixel chars)

drawBorder(x, y, w, h, color, style)Draws a border around something.
drawPixelCharacter(x, y, tl, tr, l, r, bl, br, tc, bc)Draws a drawing char.
writeCentered(text [, t [, offset]])Draws text centered.
drawBorder(x, y, w, h, color, style)

Draws a border around something. 1-box will render a 1-pixel border on the inside. 1-box-outside will render a 1-pixel border on the outside.

Parameters

  1. x number The x position to start the border
  2. y number The y position to start the border
  3. w number The width of the border
  4. h number The height of the border
  5. color number The color of the border
  6. style string The style to render in, 1-box, 2-box, 2-box-dropdown, 1-box-outside
drawPixelCharacter(x, y, tl, tr, l, r, bl, br, tc, bc)

Draws a drawing char.

Parameters

  1. x number The x position.
  2. y number The y position.
  3. tl boolean Draw top left cell.
  4. tr boolean Draw top right cell.
  5. l boolean Draw middle left cell.
  6. r boolean Draw middle left cell.
  7. bl boolean Draw bottom left cell.
  8. br boolean Draw bottom right cell.
  9. tc number Foreground color
  10. bc number Background color
writeCentered(text [, t [, offset]])

Draws text centered.

Parameters

  1. text string The text to draw.
  2. t? table The terminal to draw in.
  3. offset? number The offset
