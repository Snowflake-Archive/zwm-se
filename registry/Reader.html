Reader

Util for reading registries

new(name, fromDefaults)Creates a new RegistryReader.
reload([data])Reloads the registry.
get(key)Reads a key.
getType(key)Gets a key's type.
getEnum(key)Gets a key's enum values.
new(name, fromDefaults)

Creates a new RegistryReader.

Parameters

  1. name string The name of the registry file
  2. fromDefaults boolean If this is true, the registry will be read from the defaults folder
reload([data])

Reloads the registry.

Parameters

  1. data? table If this is provided, data will be read from the table instead of the file
get(key)

Reads a key.

Parameters

  1. key string The key to read. This is in compressed table format (e.g. One.Two.Three)

Returns

  1. any The value of the key
getType(key)

Gets a key's type.

Parameters

  1. key string The key to read. This is in compressed table format (e.g. One.Two.Three)

Returns

  1. string The type of the key (string, number, table or enum)
getEnum(key)

Gets a key's enum values.

Parameters

  1. key string The key to read. This is in compressed table format (e.g. One.Two.Three)

Returns

  1. table|nil The values of the enum.
