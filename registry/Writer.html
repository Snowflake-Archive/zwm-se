new(name)

Creates a new RegistryWriter.

Parameters

  1. name string The name of the registry file
set(key, value)

Sets a key.

Parameters

  1. key string The key to set. This is in compressed table format (e.g. One.Two.Three)
  2. value any The value to set

Returns

  1. The updated table.
