new(x, y, w, h, parent [, renderScrollbars [, visible [, defaultColor]]])

Creates a scrollbox frame via a dictionary. The below parameters are in no particular order.

Parameters

  1. x number The X position of the scrollbox frame.
  2. y number The Y position of the scrollbox frame.
  3. w number The width of the scrollbox frame.
  4. h number The height of the scrollbox frame.
  5. parent table The parent term of the scrollbox window.
  6. renderScrollbars? table A table containing an X and Y paramater, if either is true, a scrollbar will be rendered for that axis.
  7. visible? boolean If false, the scrollbox will not be rendered.
  8. defaultColor? number The default background color.

Returns

  1. Scrollbox The created scrollbox instance.
