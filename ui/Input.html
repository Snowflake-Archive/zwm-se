new(x, y, w [, onChange [, onComplete [, placeholder [, disabled [, default [, visible]]]]]])

Creates a new input object via a dictionary. The below parameters are in no particular order,

Parameters

  1. x number The X position of the input.
  2. y number The Y position of the input.
  3. w number The width of the input.
  4. onChange? function Fires when the content of the input changes.
  5. onComplete? function Fires when the input is defocused. This will fire with a method to defocusization, either "return" or "defocus"
  6. placeholder? string Text that will be rendered if the input is empty.
  7. disabled? boolean Whether or not the input can be focused
  8. default? string The default content of the input
  9. visible? boolean Whether or not the input is visible. Default is true, if this is false the input won't render, nor will it be visible.

Returns

  1. Input The net input.

setVisible(visible)

  1. visible boolean Whether or not the input is disabled.
