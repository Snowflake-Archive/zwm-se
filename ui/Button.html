new(x, y, text, callback [, disabled [, visible [, disablePadding [, colors [, term]]]]])

Creates a new button. via a dictionary. The buttons width will be #text + 4. The below parameters are in no particular order.

Parameters

  1. x number The X position of the button
  2. y number The Y position of the button
  3. text string The text that will be rendered inside the button.
  4. callback function The function that is ran when the button is clicked.
  5. disabled? boolean If this is true, the button will be grayed out and not be selectable.
  6. visible? boolean If this is false, the button will not be rendered, nor selectable.
  7. disablePadding? boolean If this is true, the text of the button will be its actual text, and not padded.
  8. colors? table A table for colors, background, clicking, focused, text, and textDisabled
  9. term? table The terminal to render the button in

Returns

  1. Button The new button.
reposition(x, y)

Repositions a button.

Parameters

  1. x number The X position to move the button to.
  2. y number The Y position to move the button to.
setText(text)

Sets the text of a button.

Parameters

  1. text string The new text of the button.
setDisabled(disabled)

Sets whether or not the button is disabled. If this is true, the button will be greyed out and not selectable.

Parameters

  1. disabled boolean Whether the button is disabled or not.
setVisible(visible)

Sets whether or not the button is visible. Note that the whole screen will need to be re-rendered to make the button disappear.

Parameters

  1. visible boolean Whether or not the button is disabled.
setFocused(focused)

Sets whether or not the button is focused.

Parameters

  1. focused boolean Whether or not the button is focused.
setCallback(The)

Sets the callback of the button

Parameters

  1. The function button callback
render([useBgRender])

Renders the button.

Parameters

  1. useBgRender? boolean If this is true, the button will be rendered with the same background color used to render it last time.

Parameters

  1. isFirst? boolean This is used for clicking with the mouse, mouse_click will make this true.
