new(objects [, triggerMethod [, colors [, dropdownStyle]]])

Creates a new context menu via a dictionary. The below parameters are in no particular order.

Parameters

  1. objects table The objects to add to the context menu.
  2. triggerMethod? table A table used to describe how the context menu will be triggered.
  3. colors? table The colors to use in the context menu.
  4. dropdownStyle? boolean If true, the top of the context menu will be chopped off.
render(x, y [, objectsOnly])

Renders a context menu.

Parameters

  1. x number The x position of the context menu.
  2. y number The y position of the context menu.
  3. objectsOnly? boolean If true, only objects will be rendered. This is primarly for internal use.
