UI Manager

An event manager to be used with UI components.

new(xOffset, yOffset)Creates a new UIManager.
getID()Gets the ID of the UI manager's instance.
addButton(button, scrollbox)Adds a button to the event manager.
addInput(input, scrollbox)Adds an input to the event manager.
addContextMenu(contextMenu, scrollbox)Adds a context menu to the event manager.
check(e)Checks events.
inject(manager)Injects the event manager into the event manager.
new(xOffset, yOffset)

Creates a new UIManager.

Parameters

  1. xOffset number X offset for mouse events
  2. yOffset number Y offset for mouse events

Returns

  1. UIManager The event manager.
getID()

Gets the ID of the UI manager's instance.

addButton(button, scrollbox)

Adds a button to the event manager.

Parameters

  1. button Button The button to add.
  2. scrollbox table If the object is contained in a scrollbox, provide the scrollbox here, so the events can react.
addInput(input, scrollbox)

Adds an input to the event manager.

Parameters

  1. input Input The input to add.
  2. scrollbox table If the object is contained in a scrollbox, provide the scrollbox here, so the events can react.
addContextMenu(contextMenu, scrollbox)

Adds a context menu to the event manager.

Parameters

  1. contextMenu ContextMenu The context menu to add.
  2. scrollbox table If the object is contained in a scrollbox, provide the scrollbox here, so the events can react.
check(e)

Checks events.

Parameters

  1. e table The event.
inject(manager)

Injects the event manager into the event manager.

Parameters

  1. manager EventManager The event manager to inject into.
