read(from, isRaw)

Reads a full registry file.

Parameters

  1. from string The registry to read from (machine or user)
  2. isRaw boolean Whether the path is raw or not

Returns

  1. table The registry table
readKey(from, key, isRaw)

Reads a key from a registry file.

Parameters

  1. from string The registry to read from (machine or user)
  2. key string The key to read. This is in compressed table format (e.g. One.Two.Three)
  3. isRaw boolean Whether the path is raw or not

Returns

  1. string The value of the key
getKeyType(from, key)

Gets a key's type.

Parameters

  1. from string The registry to read from (machine or user)
  2. key string The key to read. This is in compressed table format (e.g. One.Two.Three)
getEnumValues(from, key)

Gets a key's enum values.

Parameters

  1. from string The registry to read from (machine or user)
  2. key string The key to read. This is in compressed table format (e.g. One.Two.Three)
writeKey(from, key, data)

Reads a key from a registry file.

Parameters

  1. from string The registry to write to (machine or user)
  2. key string The key to write to. This is in compressed table format (e.g. One.Two.Three)
  3. data any What to write to the key. This must be the same as what the item is defined as.

Returns

  1. string The value of the key
update(from, to)

Upgrades an old registry to a new one.

Parameters

  1. from table The registry to read from
  2. to table The registry to write to

Returns

  1. The table that was created
