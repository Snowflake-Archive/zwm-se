events

A basic event manager, allows for adding multiple listeners to an event.

new()Creates a new event manager.
addListener(listener, callback)Adds a listener to the event manager.
check(e)Checks events.
listen([useRaw])Starts listening for events.
new()

Creates a new event manager.

Returns

  1. EventManager The event manager
addListener(listener, callback)

Adds a listener to the event manager.

Parameters

  1. listener string | table The event to listen for. If this is a table, it is iterated through as a list of events. If this is a string, it is treated as a single event, with a required callback function.
  2. callback function The callback to call when the event is fired
check(e)

Checks events.

Parameters

  1. e table A packed os.pullEvent response.
listen([useRaw])

Starts listening for events.

Parameters

  1. useRaw? boolean If this is true, os.pullEventRaw will be used rather than os.pullEvent
