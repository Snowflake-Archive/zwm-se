getPixelChar(tl, tr, l, r, bl, br)

Gets a drawing character from the cells that are set to true.

Parameters

  1. tl boolean Draw top left cell.
  2. tr boolean Draw top right cell.
  3. l boolean Draw middle left cell.
  4. r boolean Draw middle left cell.
  5. bl boolean Draw bottom left cell.
  6. br boolean Draw bottom right cell.
drawPixelCharacter(x, y, tl, tr, l, r, bl, br, tc, bc)

Draws a drawing char.

Parameters

  1. x number The x position.
  2. y number The y position.
  3. tl boolean Draw top left cell.
  4. tr boolean Draw top right cell.
  5. l boolean Draw middle left cell.
  6. r boolean Draw middle left cell.
  7. bl boolean Draw bottom left cell.
  8. br boolean Draw bottom right cell.
  9. tc number Foreground color
  10. bc number Background color
fromBlit(blit)

Converts a blit to a color number

Parameters

  1. blit string The blit char to convert

Returns

  1. number The color
selectXfromBlit(x, blit)

Selects an X position from a blit string (or just any string really)

Parameters

  1. x number The X position to pull
  2. blit string The blit string to pull from

Returns

  1. number The color
drawBorder(x, y, w, h, color, style)

Draws a border around something.

Parameters

  1. x number The x position to start the border
  2. y number The y position to start the border
  3. w number The width of the border
  4. h number The height of the border
  5. color number The color of the border
  6. style string The style to render in. The only available style is 1-box
