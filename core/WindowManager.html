Gets the system's logger.

Returns

  1. Logger The logger.
killProcess(id)

Kills a process with the specified ID.

Parameters

  1. id number The ID.
getProcesses()

Gets all running processes.

Returns

  1. Process[] The processes.
getSize()

Gets the size of the window manager.

Returns

  1. number x The size in the X axis
  2. number y The size in the Y axis
addProcess(The, options [, focused])

Creates a process

Parameters

  1. The string | function path/function to launch. Note that functions are not well supported.
  2. options ProcessOptions The options for the program.
  3. focused? boolean Whether or not the process will be focused when it is started
