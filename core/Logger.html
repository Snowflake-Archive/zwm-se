Logger

A logger good enough for ComputerCraft.

new(debug)Creates a new logger.
log(type, message, ...)Logs a message.
debug(message, ...)Logs a message as a Debug message.
info(message, ...)Logs a message as an Info message.
warn(message, ...)Logs a message as a Warning.
error(message, ...)Logs a message as an Error.
critical(message, ...)Logs a message as a Critcal Error.
dump(file)Dumps logs.
new(debug)

Creates a new logger.

Parameters

  1. debug boolean If debugging is enabled or not

Returns

  1. table The logger
log(type, message, ...)

Logs a message.

Parameters

  1. type string The type of message to log (debug, info, warning, error, critical)
  2. message string The message to log
  3. ... any Extra arguments to pass to string.format
debug(message, ...)

Logs a message as a Debug message. Extra arguments are treated as string.format arguments. If debugging is disabled when the logger was created, these will be ignored.

Parameters

  1. message string The message to log
  2. ... any Extra arguments to pass to string.format
